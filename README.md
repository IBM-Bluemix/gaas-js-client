Client for IBM Globalization-as-a-Service
===
by Steven R. Loomis

To use ( quickstart )

# install [node.js](http://nodejs.org)

# make sure node is in your PATH

# run the following
```
        npm install
        npm run
```


You can also do:
```
        set TAAS_API_URL=http://127.0.0.1:9131/translate
```
